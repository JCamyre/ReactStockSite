Steps to making Django site + React
django-admin startproject "website name"
py manage.py startapp " " (note: don't startapp for frontend, that is for React/npx)
py manage.py migrate
py manage.py 
settings.py: 
INSTALLED_APPS, add name of apps

When modifying models.py: python manage.py makemigrations todo, python manage.py migrate todo
Add stuff to admin.py when changing models

python manage.py createsuperuser

For React: 
pip install djangorestframework django-cors-headers
settings.py:
INSTALLED_APPS,     'corsheaders', 'rest_framework',
MIDDLEWARE, 'corsheaders.middleware.CorsMiddleware',
serializers.py
npx create-react-app frontend (in cd mainsite)
cd frontend, npm start
npm install @material-ui/core
